# Goal Progress - Start Recommendation Flow

- Added `/start/` as a dedicated 60-second recommendation page.
- The page asks four buyer questions: country, cooking preference, household type, and purchase priority.

- Directly supports the mission: help shoppers answer "What meal delivery service should I use?" quickly and confidently.

- Reduces reliance on large brand tables as the primary user journey.
