# Every Meal Guide CRO Backlog

## Goal

Improve affiliate click-through rate from money pages without misleading users or inflating claims.

Primary conversion event:

```text
affiliate_click
```

## P0 Tests

### 1. CTA Label Test

Current:

```text
Check current deal
```

Alternatives:

```text
See current offer
Check today's deal
Compare latest deal
```

Success metric:

Affiliate CTR by page and brand.

### 2. First Card CTA Placement

Test:

Move the first ranking card CTA higher, above pros/cons.

Why:

Many buyers already trust the first recommendation and may not need the full card.

Risk:

Could feel too pushy if it appears before enough context.

### 3. Sticky Mobile CTA Copy

Current:

```text
Top pick: {brand}
Check deal
```

Alternatives:

```text
Best first pick: {brand}
Check offer
```

Success metric:

Mobile affiliate CTR.

### 4. Deal Warning Placement

Test:

On deal pages, keep warning cards above the table versus below quick answer.

Why:

Warnings can increase trust and make the following CTA feel safer.

Success metric:

Deal page affiliate CTR and scroll depth.

## P1 Tests

### 5. Verdict Chip Copy

Examples:

```text
Safest first meal-kit test
Best no-cook routine pick
Best taste-first prepared option
```

Test shorter variants:

```text
Best first test
Best no-cook pick
Best taste pick
```

### 6. Source Notes CTA

Test:

Add a CTA inside source-note cards for verified brands.

Risk:

Too many CTAs can feel spammy.

### 7. Trust Panel Position

Test:

Move trust panel above ranking cards on brand-sensitive pages.

Success metric:

Affiliate CTR and bounce rate.

### 8. Compare Alternatives CTA

Test:

Add secondary CTA under main affiliate CTA:

```text
Compare alternatives
```

Success metric:

Internal click depth and eventual affiliate clicks.

## P2 Tests

### 9. Email Capture

Test:

Only after analytics works, offer weekly deal alerts.

Risk:

Email capture can distract from affiliate clicks.

### 10. Brand Logo/Visual Cards

Test:

Add simple brand text marks or generated visual cards.

Risk:

Need trademark-safe treatment and no fake official affiliation.

### 11. Country Filter

Test:

Add a visible country selector on global pages.

Success metric:

Internal click-through to country pages.

## Rules

1. Do not test misleading urgency.
2. Do not fake scarcity.
3. Do not hide affiliate disclosure.
4. Do not invent coupon values.
5. Do not optimize clicks at the expense of user fit.

## Measurement Notes

Every test should record:

1. Page.
2. Variant.
3. Date started.
4. Date ended.
5. Pageviews.
6. Affiliate clicks.
7. Affiliate CTR.
8. Notes.
