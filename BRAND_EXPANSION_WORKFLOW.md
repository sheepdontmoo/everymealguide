# Brand expansion workflow

## Purpose

Use this workflow to move toward 1,000+ serious brands without adding low-quality filler.

## Command

```powershell
npm.cmd run plan:brand-gaps
```

## Output

```text
seo/brand-expansion-gap-plan.csv
reports/brand-expansion-gap-plan.json
```

## Rule

Do not add a brand just to increase count. A useful brand row needs:

- brand name
- country
- category
- active or clearly labelled status
- official URL
- consumer-facing role
- affiliate/direct/contact route or research note
- source check date when verified

## Expansion priority

1. Fill P0 country/category gaps first.
2. Favor active, orderable, consumer-facing services.
3. Separate meal kits, prepared meals, high-protein, diet/specialist, frozen, family/kids, and grocery-dinner services.
4. Do not publish strong recommendations for inactive, watchlist, or unverified brands.

## Why it matters

The goal is not a giant list. The mission is to help consumers choose confidently. More brands only matter when they improve the quality of the shortlist.
